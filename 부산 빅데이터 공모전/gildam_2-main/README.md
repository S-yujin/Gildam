# gildam_2

부산 빅데이터 공모전 프로젝트 **Gildam**의 코드와 자료를 정리한 저장소입니다.

## 프로젝트 소개

Gildam은 부산 지역 데이터를 활용하여 사용자에게 유용한 정보를 제공하기 위한 빅데이터 기반 프로젝트입니다.
공공데이터와 데이터 분석을 활용하여 지역 정보를 분석하고 활용하는 것을 목표로 합니다.

## 주요 내용

* 부산 공공데이터 분석
* 데이터 전처리 및 분석
* 프로젝트 결과 코드 정리

## 폴더 구조

```
gildam_2-main
 ┣ data        # 데이터 파일
 ┣ notebook    # 데이터 분석 코드
 ┣ src         # 프로젝트 코드
 ┗ result      # 분석 결과
```

## 사용 기술

* Python
* Data Analysis
* Open Data

## 목적

부산 지역 데이터를 활용하여 의미 있는 정보를 도출하고
빅데이터 분석 역량을 향상시키기 위한 프로젝트입니다.
