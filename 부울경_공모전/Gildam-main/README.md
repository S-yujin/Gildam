# Gildam (부울경 공모전)

부울경 공모전을 위해 진행한 **Gildam 프로젝트 코드와 자료를 정리한 저장소**입니다.

## 프로젝트 소개

Gildam은 부산·울산·경남 지역 데이터를 활용하여
지역 정보를 분석하고 활용하기 위한 데이터 기반 프로젝트입니다.

## 주요 기능

* 지역 데이터 분석
* 데이터 전처리 및 활용
* 분석 결과 정리

## 폴더 구조

```
Gildam-main
 ┣ data        # 사용 데이터
 ┣ code        # 분석 및 프로젝트 코드
 ┣ result      # 분석 결과
 ┗ docs        # 프로젝트 문서
```

## 사용 기술

* Python
* Data Analysis
* Open Data

## 목적

부울경 지역 데이터를 활용하여 의미 있는 정보를 도출하고
데이터 분석 역량을 향상시키는 것을 목표로 합니다.
