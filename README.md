# Gildam

데이터 분석과 LLM을 활용한 지역 정보 프로젝트 **Gildam**의 코드와 자료를 정리한 저장소입니다.
부산 및 부울경 지역 데이터를 활용한 공모전 프로젝트를 포함하고 있습니다.

참여 공모전

* 부산테크노파크 Big Data 활용 대회
* 부울경 ICT 아이디어 경진대회

---

# 프로젝트 주제

### 길 위의 부산 이야기

부산 여행 중 방문한 지역을 기반으로 **자동으로 여행 스토리를 생성하는 앱 개발 프로젝트**

### 개발 배경

현재 여행 정보는 대부분 개인 블로그나 SNS에 의존하는 경우가 많습니다.
이로 인해 정보의 신뢰성이나 접근성이 떨어지는 문제가 있습니다.

Gildam은 **LLM과 공공 데이터를 활용하여 여행지 정보를 분석하고, 사용자의 이동 경로를 기반으로 여행 스토리를 자동 생성하는 서비스**를 목표로 합니다.

---

# 프로젝트 소개

Gildam은 지역 데이터를 기반으로 다양한 정보를 분석하고 활용하기 위한 프로젝트입니다.
공공데이터와 데이터 분석을 통해 지역 정보를 탐색하고 의미 있는 결과를 도출하는 것을 목표로 합니다.

### 부산 빅데이터 공모전

부산 지역 데이터를 활용하여 관광지 정보를 분석하고
여행 스토리 생성 기능을 구현하는 프로젝트입니다.

주요 내용

* 부산 공공데이터 활용
* 관광 명소 데이터 분석
* 여행 스토리 자동 생성

---

### 부울경 ICT 공모전

부산·울산·경남 지역 데이터를 활용하여
지역 정보를 분석하고 여행 서비스를 확장하는 프로젝트입니다.

주요 내용

* 지역 데이터 분석
* 데이터 전처리 및 활용
* 분석 결과 정리
