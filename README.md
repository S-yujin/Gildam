# Gildam

데이터 분석을 활용한 지역 정보 프로젝트 **Gildam**의 코드와 자료를 정리한 저장소입니다.
부산 및 부울경 지역 데이터를 활용한 공모전 프로젝트를 포함하고 있습니다.

---

## 프로젝트 소개

Gildam은 지역 데이터를 기반으로 다양한 정보를 분석하고 활용하기 위한 프로젝트입니다.
공공데이터와 데이터 분석을 통해 지역 정보를 탐색하고 의미 있는 결과를 도출하는 것을 목표로 합니다.

---

## Repository Structure

```
Gildam
 ┣ 부산 빅데이터 공모전
 ┃ ┗ gildam_2-main
 ┃
 ┣ 부울경 공모전
 ┃ ┗ Gildam-main
```

### 1️⃣ 부산 빅데이터 공모전

부산 지역 데이터를 활용하여 다양한 정보를 분석하고 시각화하는 프로젝트입니다.

주요 내용

* 부산 공공데이터 활용
* 데이터 분석 및 처리
* 프로젝트 결과 정리

---

### 2️⃣ 부울경 공모전

부산·울산·경남 지역 데이터를 활용하여 지역 정보를 분석하는 프로젝트입니다.

주요 내용

* 지역 데이터 분석
* 데이터 전처리 및 활용
* 분석 결과 정리

---

## 사용 기술

* Python
* Data Analysis
* Open Data

---

## 목적

지역 데이터를 활용한 분석을 통해 의미 있는 정보를 도출하고
데이터 분석 및 프로젝트 경험을 쌓기 위한 프로젝트입니다.
